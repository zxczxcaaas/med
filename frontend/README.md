# MedVault - Medical Records Frontend

Frontend for MedVault built with Next.js and FHEVM.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and fill in your contract address:
```bash
cp .env.example .env
```

3. Run development server:
```bash
npm run dev
```

## Deploy to Vercel

1. Push your code to GitHub
2. Import project in Vercel
3. Add environment variables in Vercel dashboard
4. Deploy!

The project is configured for Vercel deployment with `vercel.json`.
