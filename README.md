# Med

Store medical records with full confidentiality. Doctors can analyze without accessing personal data.

## Features

- Encrypted medical records
- Private data sharing between doctors
- Confidential analysis of medical history
- Encrypted access permission management
- Private reminders for appointments and medications
- Public decryption for emergency cases

## Setup

```bash
npm install
npm run compile
```

## Usage

Patients can add records, grant access to doctors, and share data privately. Emergency contacts can access critical data when needed.
